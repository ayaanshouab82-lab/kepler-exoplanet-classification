Machine learning classification of NASA Kepler exoplanet candidates using supervised learning
# Kepler Exoplanet Classification Using Machine Learning

This project uses NASA Kepler mission data to classify exoplanet candidates as either likely planet candidates or false positives.

## Models Used
- Random Forest
- Support Vector Machine
- Logistic Regression
- Ridge Classifier
- Decision Tree

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Results
The Random Forest model achieved the strongest performance, reaching around 80–85% accuracy.

## Project Background
This project was completed as part of my interest in astronomy, machine learning, and exoplanet detection. I used Kepler mission data to explore how supervised learning models can support the classification of exoplanet candidates.
